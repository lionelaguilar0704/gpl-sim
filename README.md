# Great Pirate Life Sim v1.0

## Grand Line Foundation Update

This is the first major foundation build.

### Major Systems
- Grand Line world map progression
- Regions with danger and travel requirements
- Career ranks and promotions
- Crew salaries, loyalty, roles, mutiny/desertion
- Combat score system
- Duel, raid, naval battle, bounty hunt, and escape systems
- Devil Fruit skill trees and awakenings
- Haki skill trees
- Businesses and yearly income
- Debt and interest
- Relationship system
- Legacy system
- Newspapers and world reaction
- Ship tiers, repairs, upgrades, and naval damage
- Mystery quest chains
- More origins, paths, islands, events, fruits, items, rivals, and polish
