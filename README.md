# Great Pirate Life Sim v0.4

A randomized pirate-era life simulator inspired by life-sim games.

## v0.4 BitLife Loop Update
- Actions no longer automatically age you.
- Added yearly energy/action system.
- Added dedicated Age Up button.
- Added more non-aging activities:
  - Train
  - Study
  - Work
  - Explore
  - Sail
  - Hunt Bounties
  - Recruit Crew
  - Crew Bonding
  - Devil Fruit Training
  - Haki Training
  - Shop / Black Market
  - Gambling
  - Rest
- Added more origins, paths, islands, Devil Fruits, items, events, and polish.
- Added mood, energy, better menus, item shop, gambling den, and more event variety.
