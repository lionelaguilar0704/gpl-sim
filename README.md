# Great Pirate Life Sim v6.5 — Corrected Power + Survival Build

This build corrects the previous issue where the new systems were not visible and older instant-death logic could still fire.

## Fixed
- Removed random instant death from danger checks and battle losses.
- Defeat now routes into survival outcomes: capture, robbery, scar, crew rescue, or Last Chance.
- Added visible Pirate Codex button on the main menu.
- Added visible Search Fruit and Weapons buttons on the main menu.
- Added weapon ownership/equip system.

## Added
- Weapon rarity tiers: Common, Uncommon, Rare, Epic, Legendary, Mythic.
- Weapon descriptions, lore, effects, movesets, special skills, and power scaling.
- Devil Fruit encyclopedia with Logia, Paramecia, Zoan, Ancient Zoan, and Mythical Zoan examples.
- Fruit inspection screen with type, rarity, description, passives, moves, and weakness.
- Danger/Recklessness meter.
- Equipped weapon display in Powers tab.
